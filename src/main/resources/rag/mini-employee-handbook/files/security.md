# Security Handbook

## Phishing Rules
- Never click suspicious links in emails.
- Report phishing attempts immediately via the *Report Phish* button in Outlook.
- IT will confirm within 24 hours if the email was malicious.

## Password Policy
- Minimum length: 12 characters.
- Must include upper & lower case, numbers, and symbols.
- Passwords expire every 90 days.
- Do not reuse old passwords.
- Multi-factor authentication (MFA) is mandatory.

## Incident Contact
If you suspect a security incident:
1. Disconnect your device from the network.
2. Call the **Security Hotline** immediately: +1 555-123-5000.
3. Email: security@company.com (from a safe device).

The Security team operates 24/7.

## Training
- Mandatory annual security awareness training.
- Phishing simulations are conducted quarterly.
